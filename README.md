# StudyQuest

Site estático desenvolvido como projeto acadêmico para organizar conteúdos de
estudo por disciplina.

## Páginas

- `index.html`: apresentação do projeto.
- `disciplinas.html`: lista de disciplinas disponíveis.
- `duvidas.html`: espaço para dúvidas e orientações.
- `matérias/`: páginas de Português, Matemática, Química e Física.

## Como abrir

Abra o arquivo `index.html` no navegador ou use uma extensão como Live Server no
VS Code para navegar pelo projeto localmente.
